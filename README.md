# Library Management System
This app allows admiistrators to upload books (ASCII text files) and users to lend an available book. The key features are:
- Upload books - Admin
- Search for books
- Request for a book
- Approve/ Reject a request - Admin

## Development Stack
- Python
  - Flask
  - SQLAlchemy
  - Celery
- SQLite
- Redis
- Vue.js
